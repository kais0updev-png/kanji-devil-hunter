バッジデザイン本来の黒い円形背景とテキストを活かすため、円形トリミングを除去：

```css
/* ポチタ専用スタイル（バッジデザインを活かす） */
.pochita .character-img {
  border-radius: 0;           /* 円形トリミング除去 */
  border: none;               /* ボーダー除去 */
  background: none;           /* 背景グラデ除去 */
  box-shadow: 0 8px 24px rgba(0, 0, 0, 0.5);
  filter: drop-shadow(0 4px 12px rgba(255, 140, 0, 0.6));
}
```

#### ホバーエフェクト
```css

  transform: scale(1.1);      /* 軽い拡大のみ */

}
```
