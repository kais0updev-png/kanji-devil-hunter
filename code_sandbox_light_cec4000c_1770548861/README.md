# 🪚 漢字デビルハンター - チェンソーマン世界観学習アプリ

アニメ『チェンソーマン』の世界観で漢字を学ぶ、ダーク&シネマティックな学習Webアプリ

## 🎯 プロジェクト概要

日本漢字能力検定10級～7級（小学1～4年生）635字の漢字を、チェンソーマンのキャラクターと共にステージクリア形式で攻略する学習アプリです。

**チェンソーマンのデザインシステムを完全再現**し、ダーク&シネマティックな世界観の中で楽しく学習できます。

## 📊 データ構造

各漢字は以下の情報を持ちます：

```javascript
{
  kanji: '山',           // 漢字本体
  onReading: 'サン',     // 音読み
  kunReading: 'やま',    // 訓読み
  radical: '山',         // 部首
  strokes: 3,           // 画数
  okurigana: '',        // 送り仮名の例
  antonyms: '',         // 対義語
  homophones: '',       // 同音異字
  compounds: '',        // 三字熟語
  grade: 1,             // 学年（1-4）
  level: 10             // 検定級（10, 9, 8, 7）
}
```

## 🚀 使用方法

### ローカルで実行

1. プロジェクトファイルをダウンロード
2. `index.html` をブラウザで開く
3. Stage 1 から順番に漢字デビルを撃破！

### 学習の流れ

1. **ホーム画面**: ステージを選択（最初はStage 1のみ解放）
2. **学習画面**: 
   - 中央に大きく漢字が表示される
   - 4つの選択肢から正しい読み方を選ぶ
   - 正解すると緑のグローエフェクト ✨
   - 不正解すると赤く振動 + 正解が表示される
3. **結果画面**: 
   - 正解率とクリア数を確認
   - 間違えた漢字を復習
   - 次のステージが解放される！

### 復習機能

- ❌ 間違えた問題は自動的にLocalStorageに保存
- 📝 ヘッダーの「REVIEW」から復習モードに入れる
- 🔄 結果画面からも「復習モード」ボタンでアクセス可能

## 🎮 使い方のヒント

### 効率的な学習方法

1. **毎日1ステージずつ** - 無理せず継続
2. **復習を忘れずに** - 間違えた問題は翌日復習
3. **完璧主義は NG** - スキップも活用
4. **進捗を楽しむ** - プログレスバーを見てモチベーション維持

### トラブルシューティング

**Q: ステージがロックされていて進めない**  
A: 前のステージを完全にクリア（64/64体撃破）すると解放されます

**Q: 復習モードが表示されない**  
A: 間違えた問題があると表示されます。最初は問題に挑戦してください

**Q: 進捗が保存されない**  
A: ブラウザのLocalStorageを有効にしてください。プライベートモードでは保存されません

**Q: モバイルで見づらい**  
A: 画面を横向きにすると見やすくなります
