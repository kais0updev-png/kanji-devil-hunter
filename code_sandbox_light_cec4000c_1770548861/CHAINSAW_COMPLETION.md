### 3. アニメーション完全実装

#### correctPulse（正解時） ✅
```css
0% → 50%: scale(1 → 1.1) + 緑グロー拡散
50% → 100%: scale(1.1 → 1) + グロー消失
時間: 0.6秒
```

#### shake（不正解時） ✅
```css
10%, 30%, 50%, 70%, 90%: translateX(-5px)
20%, 40%, 60%, 80%: translateX(5px)
時間: 0.5秒
```

#### cardAppear（ステージカード登場） ✅
```css
from: translateY(50px) + scale(0.9) + opacity(0)
to: translateY(0) + scale(1) + opacity(1)
時間: 0.5秒
順次表示: 各カード0.1秒ずつ遅延
```

#### progressPulse（プログレスバー） ✅
```css
0%, 100%: box-shadow: 0 0 10px rgba(255,0,0,0.5)
50%: box-shadow: 0 0 20px rgba(255,0,0,0.8)
時間: 2秒、無限ループ
```
